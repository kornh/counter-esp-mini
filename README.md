# counter-esp-mini
This project is to make remote counter and collect the data in a database.

The remote counter are ESP8266 ÂµC and should could on a input. The count will then be send to the backend server where it will be stored.

## Prerequisites
Please make sure the following software is installed.

Node JS: https://nodejs.org/en/download/

For Development Arduino Studio: https://www.arduino.cc/en/software is required.

## Setup project
1. Clone repository
2. Go to ./node-server
3. Run "npm install"
4. Run "npm start"
